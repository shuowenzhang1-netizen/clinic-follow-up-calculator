# Clinic Follow-Up Revenue Leak Calculator

A lightweight browser calculator for clinics to estimate missed repeat-booking
revenue caused by delayed consultation follow-ups, aftercare reminders, review
requests, and repeat-booking opportunities.

The calculator is static and runs fully in the browser. It does not store client
or patient data.

## Included

- Follow-up revenue estimate
- Recoverable revenue estimate
- Email summary CTA
- GBP 99 follow-up tracker setup CTA
- Clinic tracker dashboard preview

## Local Preview

```bash
python3 -m http.server 8765
```

Then open:

```text
http://localhost:8765/
```
